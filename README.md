## Dependency coordinates
#### Maven
```xml
<dependencies>
    <dependency>
        <groupId>org.rootservices</groupId>
        <artifactId>otter</artifactId>
        <version>1.3</version>
    </dependency>
</dependencies>
```

#### Gradle
```groovy
compile group: 'org.rootservices', name: 'otter', version: '1.3'
```
